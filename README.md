# Tenis Club de Alcoy

Sitio web estático en `HTML + CSS + JavaScript` con estas páginas:

- Inicio
- Entrenadores
- Inscripción
- Contacto

## Ejecutar en local

### Opción 1

Abrir `index.html` directamente en el navegador.

### Opción 2

Ejecutar en esta carpeta:

```powershell
python -m http.server 8000
```

Después abrir:

[http://localhost:8000](http://localhost:8000)
